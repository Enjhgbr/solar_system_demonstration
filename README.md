
# ДЕМОНСТРАЦИЯ СОЛНЕЧНОЙ СИСТЕМЫ
Симулятор солнечной системы, в которой вы можете посмотреть как по сравнению с другими планетами, движутся другие планеты.

перед вами предстаёт солнечная система, из солнца в центре, и планет вокруг неё, можно посмотреть скорость их передвижения а также их размеры.

# УСЛОВНЫЕ ОБОЗНАЧЕНИЯ:

желтый круг в центре - солнце,
оранжевый круг - меркурий,
коричневый - венера,
зеленый - земля,
красный - марс,
серый - юпитер,
серо-желтый - сатурн,
синий - уран,
розовый - нептун.
